    Tools

    5 tools MCP server endpoints — callable by any agent
    Tool Description Input Output
    ✅ forward_check
    		Validates a forward model specification against the PWM physics DAG registry. Returns pass/fail with detailed mismatch diagnostics. spec_id: str, params: dict valid: bool, diagnostics: list Docs →
    ⚙️ dispatch
    		Dispatches a reconstruction job to the PWM compute mesh (CPU/GPU). Returns a job_id for polling with result(). spec_id: str, solver: str, data_url: str job_id: str, eta_s: int Docs →
    📊 result
    		Polls a dispatched job and returns scores (PSNR, SSIM, consistency) once complete. Records accepted results on Base chain. job_id: str status: str, psnr: float, ssim: float, tx_hash: str Docs →
    🔍 benchmark_lookup
    		Queries the PWM benchmark registry by modality key or keyword. Returns spec metadata, leaderboard snapshot, and data download links. query: str benchmarks: list[BenchmarkSummary] Docs →
    📚 arxiv_fetch
    		Fetches arXiv paper metadata and abstract by arXiv ID. Optionally returns the full PDF text for review agents. arxiv_id: str, full_text: bool = False title, authors, abstract, pdf_text? Docs →

  "mcpServers": {

    }
  }
